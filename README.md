 Calculadora em Python

Este projeto é uma calculadora simples desenvolvida em Python.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de erro

---

# Como executar o script .sh

1. Abra o terminal
2. Dê permissão de execução:

chmod +x executar.sh

3. Execute:

./executar.sh

---

# Explicação do código Python

O código possui duas funções principais:

## Função calcular()

Responsável por realizar as operações matemáticas.

Recebe três parâmetros:
- a → primeiro número
- b → segundo número
- operacao → tipo da operação

Dependendo da operação escolhida (+ - * /) ela retorna o resultado.

Também possui verificação para evitar divisão por zero.

## Função main()

Responsável por:

- pedir os números ao usuário
- pedir a operação
- chamar a função calcular
- mostrar o resultado
